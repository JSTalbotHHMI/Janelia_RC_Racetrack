# ECAD

This folder is the landing zone for electrical design files for the RC racetrack project.

Typical contents for this area:

- schematics
- board layouts
- wiring diagrams
- connector pinouts
- fabrication outputs such as Gerbers, drill files, pick-and-place data, and assembly drawings

## Suggested Organization

- group source files by tool or subsystem, for example `KiCad/lighting-controller/`
- keep generated manufacturing exports in clearly named subfolders such as `fabrication/` or `release/`
- include a short local README in any subsystem folder that captures board revision notes, assembly constraints, or bring-up steps

No committed ECAD design files are present yet, so this README serves as the structure guide for future additions.
