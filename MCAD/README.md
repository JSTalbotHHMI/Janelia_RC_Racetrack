# MCAD

This folder is the landing zone for mechanical design files for the RC racetrack project.

Typical contents for this area:

- track fixtures, guides, and mounts
- enclosure designs
- printable jigs
- assembly drawings
- exported STEP, STL, DXF, or other fabrication packages

## Suggested Organization

- group source models by tool or subsystem, for example `Fusion360/camera-mount/`
- keep exported fabrication assets beside the source model in clearly named folders such as `exports/` or `print/`
- document assembly notes, hardware callouts, and revision history in a local README when a subsystem grows beyond a single file

No committed MCAD files are present yet, so this README acts as the organizational placeholder for future design work.
