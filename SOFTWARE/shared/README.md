# Shared Software Assets

This folder is intended for software assets shared across multiple tools, runtimes, or workflows in the repository.

Examples:

- protocol notes
- shared CSV or JSON formats
- sample datasets
- common configuration files
- reusable UI or media assets that do not belong to just one tool

The folder is currently empty, but it is the right place to promote data or documentation that would otherwise be duplicated between `utilities/`, tracker scripts, and system-level race tools.
